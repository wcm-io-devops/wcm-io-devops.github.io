Release 2.0.6 – 2024-12-10

Type Changes By Dynamically generate X-Vhost value. Fixes 93. twolfart

Release 2.0.4 – 2024-01-26

Type Changes By Role aem-dispatcher-cloud: Sync with default dispatcher configuration from Adobe AEM project archetype 40 to 47. Fixes 91. sseifert

Release 2.0.2 – 2023-09-08

Type Changes By Role aem-dispatcher-cloud: Use https in rewriteHomepageRedirect to avoid unnecessary http redirect. Fixes 90. trichter

Release 2.0.0 – 2023-07-03

Type Changes By Role aem-dispatcher-cloud: Add httpd.headers.htmlExpirationTimeSec parameter to control the default expiration time for text/html responses. sseifert Role aem-dispatcher-cloud: Remove httpd.headers.htmlExpirationTimeMin parameter which was interpreted incorrectly as seconds instead of minutes. Build will fail if this parameters is still in use. sseifert

Release 1.15.0 – 2023-03-17

Type Changes By Role aem-dispatcher-cloud: Introduce httpd.rewrites (ported from aem-dispatcher-ams role). Fixes 88. trichter Role aem-dispatcher-cloud: Introduce httpd.rewriteIncludes (ported from aem-dispatcher-ams role). Fixes 87. trichter Role aem-dispatcher-cloud: Introduce httpd.generateAvailableVhost and httpd.generateEnabledVhost to control vhost generation. Fixes 86. trichter Role aem-dispatcher, aem-dispatcher-ams, aem-dispatcher-cloud: Revert #83, move "security-related" deny rules back to dispatcher.filter. Fixes 85. trichter

Release 1.14.0 – 2023-01-12

Type Changes By Role aem-dispatcher, aem-dispatcher-ams, aem-dispatcher-cloud: Introduce dispatcher.filterAppend filter list. Fixes 83. trichter Role aem-dispatcher, aem-dispatcher-ams, aem-dispatcher-cloud: Block form selector with a non-empty suffix on all URLs in order to fix form validator bypass issue. Fixes 84. trichter Role aem-dispatcher, aem-dispatcher-ams, aem-dispatcher-cloud: Move "security-related" existing deny rules e.g. to prevent content grabbing to dispatcher.filterAppend filter list. Fixes 83. trichter Role aem-dispatcher-cloud: Sync with default dispatcher configuration from Adobe project archetype 36 to 39. Fixes 82. nbellack Switch to Java 11 as minimum version. sseifert

Release 1.13.0 – 2022-10-13

Type Changes By Role aem-dispatcher, aem-dispatcher-ams, aem-dispatcher-cloud: Introduce (optional) httpd.headers.permissionsPolicy to allow configuration of the Permissions-Policy header. trichter Role aem-dispatcher, aem-dispatcher-ams, aem-dispatcher-cloud: Introduce (optional) httpd.headers.featurePolicy to allow configuration of the Feature-Policy header. trichter Role aem-dispatcher, aem-dispatcher-ams, aem-dispatcher-cloud: Introduce dispatcher.passError to allow configuration of DispatcherPassError parameter. trichter Role aem-dispatcher-ams, aem-dispatcher-cloud: Don't show exact Apache/Dispatcher footer. trichter Role aem-dispatcher: Don't show exact Apache/Dispatcher version in Server footer. trichter Role aem-dispatcher-ams, aem-dispatcher-cloud fix duplicate X-Frame-Options header. trichter

Release 1.12.2 – 2022-05-11

Type Changes By Role aem-dispatcher: Add remoteip functionality to author vhosts. trichter Role aem-dispatcher: Introduce httpd.remoteIPHeader to allow configuration of the header representing the client ip. trichter Role aem-dispatcher: Fix not working condition for remoteip module introduced with 1.3.0 trichter

Release 1.12.0 – 2022-03-15

Type Changes By Role aem-dispatcher, aem-dispatcher-ams, aem-dispatcher-cloud: Make log level for mod_rewrite configurable. nbellack Role aem-dispatcher-cloud: Synch with default dispatcher configuration from Adobe project archetype 24 to 35. sseifert Role aem-dispatcher-cloud: Add back symlink of "default.host" which was removed in 1.11.2. sseifert Fix sling mappings to avoid generation of reverse mapping for "/". Because it would otherwise match every path that is not caught by sling mappings, even when they are not under a sling mapping root path. Fixes WDCONGA-21. mrozati
