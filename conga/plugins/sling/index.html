About CONGA Sling Plugin

wcm.io DevOps CONGA Plugin for Apache Sling.

Documentation

  • Usage
  • CONGA Extensions
  • Combined JSON file for defining OSGi Configurations
  • API documentation
  • Changelog

Overview

This plugin extends CONGA with:

  • Manage OSGi configuration templates in Combined JSON files or Apache Sling Provisioning file format
  • Generate OSGi configurations in .cfg.json files (as used by Apache Sling Configuration Installer Factory)
