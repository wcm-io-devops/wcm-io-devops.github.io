conga:generate-version-info

Full name:

io.wcm.devops.conga:conga-maven-plugin:1.14.7-SNAPSHOT:generate-version-info

Description:

Generates a file with information CONGA maven plugin version and CONGA plugins versions used to generate this artifact.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: generate-resources.

Optional Parameters

Name Type Since Description
<definitionTarget> String - Target path for the prepared definition files.
Default value is: ${project.build.directory}/definitions.

Parameter Details

<definitionTarget>

Target path for the prepared definition files.
  • Type: java.lang.String
  • Required: No
