Ansible Automation for AEM

wcm.io DevOps provides a set of Ansible roles to automate infrastructure setup and configuration management for Adobe Experience Manager (AEM).

The Ansible roles are optimized to be used together with CONGA, but some can also be used standalone.

A good starting point for setting up AEM projects for deployment automation with CONGA and Ansible is using the
wcm.io Maven Archetype for AEM Configuration Management.

If anything goes wrong, have a look at Debugging Ansible Automation for AEM.
